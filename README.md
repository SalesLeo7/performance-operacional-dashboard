# 📊 Acompanhamento Outbound

Sistema de acompanhamento de performance de order. KPI's de OTIF, Processamento, OTD. 

## Sobreo o projeto

Este projeto tem como objetivo acomanhar a performance dos pedidos, estou medindo o leed time do processo desde o recebimento do pedido até a entrega no destinatário.  

## 🧠 Funcionalidades

- 📥 Importação de dados (CSV, Excel, etc.)
- 🧹 Tratamento e limpeza de dados
- 📅 Padronização de datas
- 📊 Geração de métricas e indicadores
- 📈 Análise de performance
- 🔎 Identificação de inconsistências

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
- Knime
